# Data dictionary — Cornerstone prospect dataset

Generated by `pipelines/build_dataset.py`. Three tables land in
`data/processed/` as both `.parquet` (full fidelity, incl. list columns) and
`.csv` (list/trajectory columns dropped):

| File | Grain | Description |
|------|-------|-------------|
| `prospect_features.{parquet,csv}` | one row per drafted player | Pre-draft features only (no leakage). |
| `realized_outcomes.{parquet,csv}` | one row per drafted player | Realized NBA outcomes + tier label. |
| `prospects.{parquet,csv}` | one row per drafted player | The two joined on `player_id`. |
| `dybantsa.{parquet,csv}` | single row | AJ Dybantsa, same schema as `prospect_features`. |

Universe: every pick of the **2003–2022** NBA drafts (`config.DRAFT_YEARS`).
Outcomes have had at least ~3 seasons (most far more) to mature by 2025-26.

draft-relevant sample and avoids mixing freshman/senior production.

### Outcome tier (label)

Tier is assigned from **career VORP**, the best single public summary of total
career value, with a games floor (`config.MIN_GAMES_FOR_NON_BUST`):

| Tier | Rule (career VORP) |
|------|--------------------|
| `superstar` | ≥ 25 |
| `all_star` | 10 – 25 |
| `starter` | 3 – 10 |
| `rotation` | 0 – 3 |
| `bust` | < 0, **or** < 50 career games |

Thresholds live in `config.VORP_TIER_BINS` and are intentionally easy to revise.

## Join & disambiguation assumptions

- **Join key is the BBRef NBA `player_id`**, parsed from the draft-page link —
  not the player's name — so accents/suffixes (Jr., III) and name collisions
  don't break the join. The college page is reached via the BBRef→SRef link on
  the player page, again id-based.
- Picks with no BBRef player link (a handful of never-signed international
  picks) are skipped and counted in the build summary.
- International / prep prospects usually have **no college page**;
  `has_college_stats=False` and college columns are null. Later phases impute or
  branch on this flag rather than dropping these rows.

## Known limitations

- **Survivorship / era effects:** advanced metrics (BPM/VORP) shift across eras;
  the back-test (Phase 4) controls for draft slot as a baseline.
- **Listed measurements** (height/weight) are NBA-listed, not combine-measured.
- **Outcome maturity:** 2020–2022 classes are still developing; treat their
  tiers as provisional. The bulk of the universe (2003–2019) is mature.
